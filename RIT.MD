# **DIVISIONS**
**WEB DEVELOPMENT**
Fokus pada perancangan, pembangunan, dan pemeliharaan website serta aplikasi berbasis web, mencakup UI, programming, dan database.


**GAME DEVELOPMENT**
Mempelajari pengembangan game dari konsep hingga rilis, termasuk game mechanics, level design, dan trategi untuk membangun interaksi pemain yang kuat.

**INTERNET OF THINGS**
Eksplorasi perangkat cerdas dengan sensor dan konektivitas internet untuk membangun sistem otomatis berbasis data.

**CYBER SECURITY**
Mempelajari strategi perlindungan sistem, jaringan, dan aplikasi dari ancaman siber.


## GAME COMMUNITY

**Gaming Enthusiast**
